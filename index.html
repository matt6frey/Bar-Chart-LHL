Bar-Chart Library Demo

Default Charts

Default Charts can be implemented by using the function drawBarChart(Array, Object, Element). For example, this chart was created using the code:

drawBarChart([10, 3, 15 ,16 ,7], { title: 'A Default Chart', width: 500 }, '.bar-chart')

Many of the elements within the chart are customizable.

The Trouble with TPS Reports

The TPS Reports chart, shows some of the custom styling you can add to your chart.

It was generated using this code:

drawBarChart([10,13 ,16], { title: 'TPS Reports', titleSize: 1.25, titleColor: "darkred", chartColor: 'ghostwhite', barColors: ['#c04d4d', '#894343', 'darkred'], xLabels: ['January', 'February', 'March'], height: 400, width: 700, }, '#custom')

Stacked Charts

Stacked charts can be created by giving the data parameter array values instead of single digits.

Check out this example code:

drawBarChart([[2,2,3], [6,4,5], [8,4,6], [4,2,3], [1,2,1]] , { title: "Specie Composition", xLabels:['Central MixedWood', 'Dry Mixedwood', 'Boreal Highlands', 'Montane', 'Kazan Uplands'], barColors: ['lightgreen','green','forestgreen'], yAxis: "Population %", xAxis: "Subregion", chartBorder: "#666"} , 'div[data-id="my-chart"]')

Summary

The Bar charts library aims to speed up coding charts and provide a simple way to add them to any content.
